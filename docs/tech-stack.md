# Tech Stack: Postie

## Core Framework
- Next.js 16 (App Router, caching components, React Compiler)
- React 19

## UI and Styling
- Tailwind CSS 4
- HeroUI v3 beta (UI component library)

## Authentication and Database
- Better Auth (integrated with Supabase)
- Supabase (BaaS for auth, DB, storage)
- Prisma (ORM for Supabase queries)

## Validation and Forms
- Zod v4 (schema validation on client/server)
- @conform-to/react (form validation between client/server)
- @t3-oss/env-nextjs (environment variable validation)

## Architecture and Tools
- Architecture: Feature-Sliced Design
- Package Manager: pnpm
- Code Formatter/Linter: Biome

## Additional
- TypeScript (strict mode)
- Local Storage (for variables, with Supabase sync)
- State Management: React Context
